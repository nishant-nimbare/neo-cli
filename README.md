# neo-cli
bash script for my day-to-day tasks



### to use script

make scripts executable
```
chmod +x ./neo
```

and then restart
```
neo restart
```

### to access from anywhere
add 
```
PATH="{path_to_this_folder}/neo-cli:$PATH"
```
at the end your of `~/.profile` file


## usage
```
neo help
```
